## Assistant Response
```text
- 당월 초일(t)과 전월 초일(t-1) 기준으로 각 종목 i의 보유 수량 S를 비교한다(조건: S(t-1,i) < S(t,i)).   
- 조건을 만족하는 종목에 대해 분자에 “수량 증가분” (S(t,i)−S(t-1,i))을 잡고, 이를 종가의 평균값 ((P(t,i)+P(t-1,i))/2)으로 나눈 값을 계산한다.   
- 위 값을 조건을 만족하는 종목들에 대해 합산한 뒤, 분모에 (전월·당월 주식평가액 합의 평균) {(EV(t)+EV(t-1))/2}을 적용하여 100을 곱한다. 이 결과가 “추정 주식매수회전율”이다.   
- 당월의 추정 주식매매회전율(TR_t)은 추정 주식매도회전율(TR_t^S)과 추정 주식매수회전율(TR_t^B)의 합으로 산출한다(즉, 수량 증가분은 TR_t^B로 반영되어 TR_t에 더해진다).
```
