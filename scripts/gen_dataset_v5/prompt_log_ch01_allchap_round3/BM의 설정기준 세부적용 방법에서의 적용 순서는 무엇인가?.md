BM 설정기준의 세부적용 방법은 펀드의 BM 종류/통화처리/적용시점(시차)/수익률 산정 방식 순으로 정해 적용합니다. 

- 먼저 유형 BM vs 펀드 BM을 결정합니다.  
  국내는 테마펀드·기타인덱스를 제외하고 별도 펀드 BM 없이 유형 BM을 적용하고, 해외는 펀드 BM을 적용합니다. 해외는 설정 당시 운용사가 희망한 BM을 우선 적용하되, 제로인이 보유하지 않으면 희망 BM과 가장 유사한 BM으로 전환하며, BM이 없거나 유사 BM이 없으면 유형 BM으로 전환합니다. 

- 다음으로 해외펀드는 환헷지 여부에 따라 지수 통화를 구분합니다.  
  환헷지를 하면 로컬 지수, 환헷지를 하지 않으면 원화환산 지수를 적용하되, 원화환산 지수는 적극적으로 밝힌 경우에만 인정합니다. 

- 이어서 BM 적용일을 정합니다.  
  국내는 “펀드 기준일의 BM 적용일 = 기준일-1일”로 두고, 해외는 투자지역별로 “아시아~호주(중국~일본, 호주 등)는 당일(단 FoFs는 국내기준 1영업일 Delay, 인도는 당일-1)) / 이외 지역은 당일-1(단 FoFs는 국내기준 1영업일 Delay)”로 적용합니다.

  주식은 Capital Return 지수를, 채권은 Total Return 지수(단 BM 수익률의 90%만 적용)를 사용합니다.
