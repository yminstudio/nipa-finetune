## Assistant Response
```text
- 델타노멀(Delta Normal) 전제에 따라 유의수준 α와 보유기간 T(1주)를 확정한다.   
- KOSPI 변동성은 EWMA로 계산한다(σ²t+1|t = λσ²t|t−1 + (1−λ)r).   
- 계산된 KOSPI 변동성을 포트폴리오 변동성 V(Pt)의 함수에 반영해 V(Pt)를 산출한다.   
- 최종적으로 손실 임계치는 VaR = α √(T · V(Pt))로 산출한다.
```
