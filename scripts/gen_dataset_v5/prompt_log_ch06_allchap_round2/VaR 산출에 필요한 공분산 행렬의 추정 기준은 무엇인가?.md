## Assistant Response
```text
- VaR(Delta Normal Method) 산출에서 공분산 행렬 추정 기준으로 제시되는 것은 KOSPI 변동성(분산) 추정이며, KOSPI 변동성은 EWMA(Exponentially Weighted Moving Average)를 사용해 측정합니다.   
- EWMA 분산 추정은 다음 식을 기준으로 합니다: \( \sigma^2_{t+1|t}=(1-\lambda)\sum_{t=1}^{T}\lambda^{t-1}r_t^2=\lambda\sigma^2_{t|t-1}+(1-\lambda)r \).
```
