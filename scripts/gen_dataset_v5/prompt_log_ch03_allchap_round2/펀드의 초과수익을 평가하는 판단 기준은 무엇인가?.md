- BM대비 초과수익률 = 자산배분효과 + 자산배분타이밍효과 + 종목선정효과   
- 확실성등가(CE) : CE^F = 평균 주간 로그수익률 − λ^F × (주간 로그수익률 표준편차)^2    
- 기간별 초과성과(CE) : CE_{i,j} = R_{i,j} − λ_{i,j} × σ_{i,j}^2   
- ZI(초과성과 표준화) : ZI_{i,j} = (CE_{i,j} − Peer Group 내 CE 평균) / Peer Group 내 CE 표준편차   
- Modified Sharpe Ratio : MS^F = (주간 로그수익률 평균 − 무위험이자율 주간 로그수익률 평균) / (σ^F)^{ |(평균−무위험)|/(평균−무위험) }   
- 정보비율(IR) : IR^{F,M} = (주간 로그수익률 평균 − 시장(또는 BM) 주간 로그수익률 평균) / 추적오차(Tracking Error)
