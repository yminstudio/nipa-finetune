# v8 round2 final-export 품질 평가

## 평가 대상

- 모델 경로: `llm_model_full/gpt-oss-20b-seed-v8-round2-full-ft/final-export`
- 검증 리포트: `tests/log/v8_round2_full_ft_final_export_report.md`
- 검증 설정: `max_new_tokens=512`, `do_sample=False`

## 요약

- `final-export`는 실제로 재로딩 가능했고 validator 결과가 `success`였다.
- 샘플 5건 모두에서 Harmony `final` 답변 추출이 정상 동작했다.
- 내부 추론 누출 없이 사용자에게 보여줄 최종 답변만 안정적으로 남았다.
- `512` 토큰 기준으로 이번 샘플에서는 문장 중간 잘림이 관찰되지 않았다.

## 품질 판단

### 1. 형식 안정성

- `Raw Generation` 끝에 `<|return|>`이 붙고, `Generated Final Answer`에서는 해당 제어 토큰이 제거된 최종 답변만 남는다.
- 즉, Harmony 형식 적합성은 현재 산출물 기준으로 양호하다.

### 2. 내용 적합성

- 질문들은 모두 "충분성, 비교성, 지속성" 정의를 묻는 변형 질문인데, 답변은 세 항목을 빠짐없이 설명하고 마지막에 요약까지 제공한다.
- 학습 데이터의 의도에서 크게 벗어나는 확장이나 잡음은 보이지 않는다.

### 3. 완결성

- 이전 `256` 토큰 기준에서 문제였던 중간 잘림은 이번 `512` 토큰 검증 샘플에서는 재현되지 않았다.
- 적어도 현재 샘플 세트에서는 답변 단락과 요약 문장까지 모두 마무리된다.

### 4. 한계

- 5개 샘플이 모두 같은 개념의 paraphrase 질문이라 답변 골격도 거의 동일하다.
- 따라서 이번 평가는 "형식 안정성"과 "완결성"을 확인하는 데는 충분하지만, 베이스 모델 대비 일반화 향상까지 결론내리기에는 범위가 좁다.

## 결론

현재 확보된 `round2 final-export`는 "운영적으로 사용 가능한 저장 산출물"이며, 샘플 기준으로는 형식 안정성, 답변 완결성, 데이터 적합성이 모두 양호하다. 다만 베이스 모델 대비 향상폭 평가는 별도 비교 실행이 필요하다.
